## 데이터소개
  - 슈퍼마켓의 매출과 매장의 크기, 일일 이용자수의 관계를 분석한 데이터이다.
    - Store ID : (index) 매장 번호
    - Store Area : 매장 크기
    - Items_Available : 판매가능한 품목 수  
    - Daily_Customer_Count : 일일 평균 방문자 수
    - Store_Sales : 매출
